# file-management-helper-scripts
A small collection of helper scripts for file management and solving some annoying problems I had... like .webp images (grrrr). Written in Python and platform independent where possible. 

# How to use
Should be quasi-nothing.
- Requires Python 3 (any version should be good)
- For the webp converter, make sure you have whatever decoder is needed for your OS and that it's in PATH or, on Windows, in a local directory titled "libwebp"
- Run via terminal with one or more pathes as arguments. Optionally use the "-r" flag to recursively process all subdirectories of given pathes.
